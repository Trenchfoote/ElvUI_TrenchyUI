# TrenchyUI Changelog

## v1.3.5

### New Features
- Damage Meter: Death recap drilldown — click a player in Deaths mode to see the events leading to their death with spell icons, HP% remaining, damage values, and killing blow / avoidable indicators
- Skins: World Quest Tab addon skin — matches ElvUI styling for the tab, list, filters, settings panel, and quest rows

### Bug Fixes
- Cooldown Manager: Fixed essential and utility viewers not appearing for users who previously had them set to Hidden in Blizzard's Edit Mode
- Damage Meter: Fixed NPC companion names (e.g. Valeera in delves) showing as '?' during combat

### Improvements
- Damage Meter: DPS and total damage values are now aligned in separate columns for consistent bar readability
- Damage Meter: Deaths mode now shows death timestamp instead of empty values
- Damage Meter: Bar tooltips now resolve party member info during combat using secret-safe lookups
