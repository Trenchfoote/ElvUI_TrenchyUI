# TrenchyUI Changelog

## v1.6.2

### New Features
- QoL: Mute Annoying Sounds now also covers Ban-Lu (Monk Class Mount).
- Unit Frames: New `[tui-mplusforces]` tag — shows what percent of Mythic+ forces an enemy is worth.
- Unit Frames: New Ironfur tracker bar (Guardian only). Inspired by the addon "Simple Ironfur Tracker", but baked into ElvUI.

### Bug Fixes
- Skins: More fixes for the Platy skin.

### Improvements
- Cooldown Manager: Removed the Reminders feature — out of scope for TUI. Standalone addons like BuffReminders are a much better fit.
- Damage Meter: Bar values now show one-decimal precision (e.g. "1.2K" instead of "1K").
- Nameplates: Removed all Platynator options...integration wasn't great so scrapping it for now.
- QoL: Trenchy minimap button bar now picks up Narcissus.
