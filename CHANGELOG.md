# TrenchyUI Changelog

## v1.3.5

### Bug Fixes
- Cooldown Manager: Fixed essential and utility viewers not appearing for users who previously had them set to Hidden in Blizzard's Edit Mode
- Damage Meter: Fixed NPC companion names (e.g. Valeera in delves) showing as '?' during combat

### Improvements
- Damage Meter: DPS and total damage values are now aligned in separate columns for consistent bar readability
- Damage Meter: Bar tooltips now resolve party member info during combat using secret-safe lookups
