# TrenchyUI Changelog

## v1.6.2

### New Features
- QoL: Mute Annoying Sounds now also covers Ban-Lu (Monk Mount).
- Unit Frames: New `[tui-mplusforces]` tag — shows what percent of Mythic+ forces an enemy is worth (only on attackable enemies during an active key).

### Bug Fixes
- Skins: Fixed yellow artifacting around the tabs in the Platynator customisation window.

### Improvements
- Damage Meter: Bar values now show one-decimal precision (e.g. "1.2K" instead of "1K").
- Nameplates: Removed the Platynator target / mouseover class-color overrides — coverage was inconsistent across Platynator's internal display swaps. Use Platynator's native color settings instead.
- QoL: Trenchy minimap button bar now picks up Narcissus when Leatrix Plus isn't installed.
